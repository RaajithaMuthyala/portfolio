CKD Risk Prediction

Statistical Modeling & Clinical Parameter Analysis in R

Project Overview

This project explores statistical modeling for Chronic Kidney Disease (CKD) risk prediction using clinical blood parameters. All analysis and modeling were conducted using R, with a focus on reproducibility, interpretability, and healthcare applicability.

Workflow & Statistical Concepts

• Performed Exploratory Data Analysis (EDA) to assess missing values, distributions, and outliers using ggplot2 and dplyr.

• Handled imputation and normalization with tidyr and caret preprocessing functions.

• Conducted bivariate analysis to assess correlation between variables (e.g., urea, glucose, hypertension) and CKD status.

• Built logistic regression models and classification trees using glm() and rpart().

• Evaluated model performance with k-fold cross-validation using caret and visualized metrics with ROCR and pROC.

R Packages Used

dplyr, ggplot2, tidyr, caret, pROC, rpart, ROCR, corrplot, e1071

Clinical Relevance

CKD is often underdiagnosed until late stages. This project emphasizes early identification using accessible clinical markers such as blood pressure, urea, creatinine, and glucose levels. Statistical models like logistic regression and decision trees offer explainability — making them suitable for practical clinical deployment.

Insights

Key predictors were visualized using variable importance plots and coefficient interpretation. Features such as proteinuria, hypertension, and serum creatinine consistently showed strong association with CKD status. Multicollinearity was checked using correlation matrices and variance inflation factors (VIF).

Takeaways

This project demonstrates the power of classical statistical models in clinical risk prediction. By combining EDA, domain knowledge, and reproducible R pipelines, we developed a CKD screening framework that is transparent and scalable for real-world use.
