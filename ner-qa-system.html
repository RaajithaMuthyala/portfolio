NER + QA System

Biomedical Named Entity Recognition & Question Answering with PubMed Data

Project Overview

This project focuses on developing a biomedical question-answering system based on entity recognition from PubMed abstracts. We applied Named Entity Recognition (NER) to extract clinically relevant terms and fine-tuned a T5-small model for generating evidence-based answers from biomedical literature.

Key Features

• Extracts biomedical entities (e.g., diseases, chemicals, procedures) using SciBERT + UMLS mapping.

• Fine-tuned T5-small model generates answers to domain-specific queries using extracted content.

• Fully interactive Streamlit interface for exploration and query-based retrieval.

• Supports real-time querying of biomedical knowledge grounded in PubMed.

Technologies Used

NER: SciBERT, SpaCy, UMLS

QA Model: T5-small fine-tuned on domain-specific abstracts

Interface: Streamlit for QA interaction

Dataset: PubMed Abstracts, curated via MeSH and SDOH filters

Explainability & Insights

Each answer is backed by the exact PubMed abstract that informed the response. Entity extraction provides transparency into which parts of the text contribute to model output, helping users trace evidence. This approach improves trustworthiness and enables domain-specific validation.

Streamlit UI

The interface allows users to paste biomedical text or enter free-text questions. Extracted entities are highlighted, and selecting one generates a contextual answer from relevant PubMed content. This makes exploration both interactive and evidence-grounded.
