CVD Risk Analysis

Cross-Sectional Study using Python & ML Techniques

Project Overview

This cross-sectional project investigates significant health indicators influencing cardiovascular disease (CVD) risk. Using health screening data and Python, the study includes statistical testing, data visualization, and machine learning modeling to explore predictive relationships.

Key Features

• Performed outlier detection using boxplots, Z-scores, IQR method, Isolation Forest, and LOF.

• Normality tested using Shapiro-Wilk, Q-Q plots, and Kolmogorov–Smirnov tests.

• Logistic regression and ML models developed: Random Forest, SVM, Decision Tree.

• Categorical association tested using Chi-square; correlation matrix with heatmaps.

Technologies Used

Languages: Python

Libraries: pandas, numpy, matplotlib, seaborn, sklearn, scipy, statsmodels

Models: Logistic Regression, Random Forest, SVM, Decision Tree

Tools: Jupyter Notebook, Tableau

Model Insights

The SVM model outperformed others with highest accuracy and F1-score. Logistic regression revealed strong associations for age, systolic/diastolic BP, cholesterol, and BMI. Behavioral factors like physical activity and alcohol showed weaker but statistically significant associations.

Visual Findings

Visualizations showed age group 50–60 had the highest CVD prevalence. Obese individuals and those with elevated systolic BP had higher incidence. Gender-wise, males were more affected. Physical activity showed limited protective effect in the dataset.

Statistical Methods

• Shapiro-Wilk, KS-test for distribution assumptions

• Spearman & Kendall rank correlation for monotonic relationships

• Chi-square test for categorical variable impact

Conclusion

The project validates existing research on CVD risk and showcases the utility of interpretable ML models in early screening. Despite modest model accuracy, the framework sets a foundation for scalable, real-world cardiovascular risk prediction tools.
